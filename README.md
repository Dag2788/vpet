# vpet

Requires npm
Requires Node v14 or higher.

My workaround for Node compatibility issues: 
nvm install v18.15.0
nvm use v18.15.0

Running the app:
1) cd Vpet
2) npx expo start

  Android:
  1) Connect to same wifi
  2) Install Expo App
  3) Scan QR Code

  IOS:
  1) Scan QR Code

  WEB:
  1) Press 'w'
